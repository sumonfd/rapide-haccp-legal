Politique de confidentialité — RAPIDE HACCP

Dernière mise à jour : 21 juillet 2026

RAPIDE HACCP (« l'application ») est édité par Md Musfiqul Alam, développeur individuel (« nous », « notre »). Cette politique explique quelles données sont collectées, comment elles sont utilisées, et vos droits en application du RGPD (UE 2016/679) et de la loi française Informatique et Libertés.

1. Données que nous collectons

L'application stocke vos données localement sur votre appareil iOS et, pour certaines fonctionnalités (capteurs sans fil, IA cloud), peut les transmettre à des services tiers. Aucune donnée n'est envoyée vers nos serveurs.

1.1 Données stockées localement (sur votre iPhone)

  • Compte utilisateur : adresse e-mail, nom de l'établissement, secteur d'activité, numéro SIRET, nom du responsable.
  • Mot de passe : haché avec sel cryptographique (SHA-256), jamais stocké en clair.
  • Données HACCP : équipements, relevés de température, réceptions, plans de nettoyage, formations, lots de production, étalonnages, actions correctives, etc.
  • Étiquettes & fournisseurs : noms de produits, dates DLC, allergènes, coordonnées des fournisseurs.
  • Paramètres : préférences de langue, type d'entreprise, logo personnalisé.

Toutes ces données restent sur votre appareil et sont chiffrées par le système de stockage iOS (SwiftData/Core Data + Data Protection).

1.2 Données stockées dans le trousseau iOS (Keychain)

  • Clé API MOCREO
  • Clé API DeepSeek (si activée)
  • Clé API OpenAI / ChatGPT (si activée)
  • Clé API Anthropic / Claude (si activée)
  • Clé API Google Gemini (si activée)

Ces clés sont chiffrées par iOS et accessibles uniquement à l'application.

1.3 Données envoyées à des tiers (uniquement si vous activez la fonctionnalité)

  • API MOCREO (mocreo.com) : utilisée pour récupérer les températures de vos capteurs sans fil. Les requêtes envoient uniquement votre clé API et le numéro de série du hub.
  • API DeepSeek / OpenAI / Claude / Google Gemini : si vous activez un assistant IA cloud, les questions que vous posez sont transmises à l'API du fournisseur choisi pour générer une réponse. Aucune donnée HACCP de votre établissement n'est partagée automatiquement — uniquement le texte que vous saisissez dans le chat.
  • Apple Push Notifications : utilisé pour les rappels HACCP locaux (aucune donnée personnelle envoyée à Apple).

1.4 Données non collectées

Nous ne collectons pas :

  • Vos données HACCP, recettes, ou informations clients à des fins commerciales
  • Votre position GPS
  • Vos contacts
  • Vos photos en dehors de celles que vous choisissez d'importer pour OCR/étiquette
  • Vos identifiants publicitaires (IDFA)
  • Aucun cookie, aucun traceur

3. Permissions iOS demandées

  • Bluetooth — pour communiquer avec les sondes Testo et capteurs BLE
  • Caméra — pour scanner les codes-barres et les étiquettes (OCR)
  • Photos — pour importer/enregistrer les photos d'étiquettes produits
  • Notifications — pour les rappels HACCP automatiques (9 h / 18 h)
  • Réseau local — pour communiquer avec votre hub MOCREO en Wi-Fi

Vous pouvez révoquer chacune de ces autorisations à tout moment via les Réglages iOS.

4. Durée de conservation

  • Données HACCP : conservées sur votre appareil aussi longtemps que vous utilisez l'application. La traçabilité HACCP exige une conservation minimum 3 ans selon l'Arrêté du 21 décembre 2009.
  • Compte utilisateur : supprimé lorsque vous vous déconnectez et désinstallez l'application.
  • Clés API : supprimées lorsque vous les retirez via Réglages ou désinstallez l'application.

5. Partage et transferts

Nous ne vendons jamais vos données. Aucune donnée n'est partagée avec des annonceurs ou des courtiers de données.

Les services tiers que vous activez explicitement (MOCREO, DeepSeek, OpenAI, Anthropic, Google Gemini) reçoivent uniquement les données nécessaires à leur fonctionnement. Consultez leurs politiques de confidentialité respectives :

  • MOCREO : mocreo.com/privacy
  • DeepSeek : platform.deepseek.com/privacy
  • OpenAI : openai.com/privacy
  • Anthropic : anthropic.com/legal/privacy
  • Google Gemini : ai.google.dev/gemini-api/terms

6. Vos droits (RGPD)

  • Accès : consulter toutes vos données via l'application (Réglages → Exporter les données)
  • Rectification : modifier vos données via l'application
  • Effacement : supprimer votre compte ou désinstaller l'application
  • Portabilité : exporter vos données HACCP en CSV (Réglages → Exporter)
  • Opposition : refuser les services tiers (l'application fonctionne sans IA cloud, sans MOCREO si vous saisissez les températures manuellement)
  • Réclamation : déposer une plainte auprès de la CNIL (www.cnil.fr)

Pour exercer ces droits, contactez : rapidehaccp@outlook.fr

7. Sécurité

  • Mots de passe hachés (SHA-256 + sel aléatoire 32 octets)
  • Clés API stockées dans le trousseau iOS chiffré
  • Base de données locale protégée par le chiffrement iOS Data Protection
  • Communications réseau en HTTPS uniquement (TLS 1.2+)
  • Aucun serveur cloud sous notre contrôle qui stocke vos données

8. Enfants

L'application est destinée aux professionnels de l'alimentation et n'est pas conçue pour les enfants de moins de 16 ans. Nous ne collectons sciemment aucune donnée d'enfants.

9. Modifications

Nous pouvons mettre à jour cette politique. La date « Dernière mise à jour » en haut indique la version courante. Les modifications majeures vous seront notifiées dans l'application.
