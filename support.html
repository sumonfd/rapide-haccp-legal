Questions fréquentes

Comment créer un compte ?
Ouvrez l'application et suivez les instructions d'inscription, ou connectez-vous directement avec « Se connecter avec Apple ».

L'assistant IA est-il obligatoire ?
Non. L'assistant IA est entièrement facultatif. L'application est pleinement fonctionnelle sans lui. Si vous l'activez, vous devez fournir votre propre clé API (Anthropic, OpenAI, Google ou DeepSeek), sauf pour Apple Intelligence qui fonctionne sans configuration.

Mes données sont-elles envoyées à un serveur ?
Non. Vos données HACCP restent sur votre appareil et se synchronisent uniquement via votre compte iCloud privé. Consultez notre politique de confidentialité pour plus de détails.

Comment signaler un bug ou demander une fonctionnalité ?
Envoyez-nous un e-mail à rapidehaccp@outlook.fr avec une description et, si possible, une capture d'écran.

Comment supprimer mon compte ou mes données ?
Vous pouvez supprimer votre compte directement depuis l'application (Réglages), ou nous écrire pour toute demande liée au RGPD.
